# Domande e risposte di Sistemi embedded

Questo repo contiene le possibili [domande](domande.md) chieste durante le sessioni d'orale dell'esame di Sistemi Embedded.

Le [risposte](risposte.md) **non** sono risposte ufficiali, sono state redatte da studenti e sono state modificate **da me**. Possono essere presenti errori.

Alcune sono decenti altre MOLTO meno, a voi capire quali e come integrarle eventualmente. **CAVEAT EMPTOR**

## Funzionamento script

Eseguire `estraiDomande.sh` per vedere una domanda. Vengono salvate in un file txt per evitare ripetizioni. Al finire delle domande il file txt viene cancellato e si riparte dallo stato iniziale.
